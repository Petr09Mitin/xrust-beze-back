Вопрос:
Как мне получить дату и время в python?

Ответ:
Использовать datetime:

>>> import datetime
>>> now = datetime.datetime.now()
>>> now
datetime.datetime(2009, 1, 6, 15, 8, 24, 78915)
>>> print(now)
2009-01-06 15:08:24.789150
Только для отображения времени на часах без указания даты:

>>> now.time()
datetime.time(15, 8, 24, 78915)
>>> print(now.time())
15:08:24.789150
Чтобы сэкономить на вводе текста, вы можете импортировать datetime объект из datetime модуля:

>>> from datetime import datetime
Затем удалите префикс datetime. из всего вышеперечисленного.



Вопрос:
Для чего в проекте нужен docker-compose.yaml?

Ответ:
Docker-compose.yml — это файл конфигурации для Docker Compose.

Это позволяет одновременно развёртывать, объединять и настраивать несколько Docker-контейнеров. «Правило» Docker заключается в том, чтобы передавать каждый процесс в отдельный Docker-контейнер.

Возьмём, к примеру, простое веб-приложение: вам нужен сервер, база данных и PHP. Таким образом, вы можете создать три Docker-контейнера с Apache2, PHP и MySQL.

Преимущество Docker Compose заключается в простоте настройки. Вам не нужно писать множество команд в Bash. Вы можете заранее задать их в файле docker-compose.yml:

db:
    image: mysql
    ports:
        - "3306:3306"
    environment:
        MYSQL_DATABASE: example_db
        MYSQL_USER: root
        MYSQL_PASSWORD: rootpw
php:
    image: php
    ports:
        - "80:80"
        - "443:443"
    volumes:
        - ./SRC:/var/www/
    links:
        - db
Как вы можете видеть в моём примере, я определяю переадресацию портов, тома для внешних данных и ссылки на другой контейнер Docker. Это быстро, воспроизводимо и не так уж сложно для понимания.

Формат файла Docker Compose официально определён, что позволяет запускать файлы docker-compose.yml с помощью чего-то другого, кроме Docker, например Podman.
